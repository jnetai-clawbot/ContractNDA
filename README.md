# ContractNDA — Contract & NDA Reviewer

**Job 56** | **Package:** com.jnetai.contractnda | **Repo:** https://github.com/jnetai-clawbot/ContractNDA

## What It Does
Review contracts and NDAs clause-by-clause. Detect red flags automatically, score risk levels, and track which clauses you've reviewed.

## Features
- Contract management (NDA, employment, service, lease, etc.)
- Clause-by-clause review with risk scoring
- Automatic red flag detection (unusual liability, non-compete scope, etc.)
- Risk level assessment (Low/Medium/High)
- Review tracking per clause
- Data export as JSON

## Build
- Android API 26+, Kotlin, Room, KSP, ViewBinding + XML, Material3 Dark
- GitHub Actions CI with release APK
